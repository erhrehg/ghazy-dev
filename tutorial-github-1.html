Cara Upload Project ke GitHub untuk Pemula

Ghazy1 April 20267 menit bacaYogyakarta

GitHub adalah platform untuk menyimpan dan berbagi kode. Kalau kamu sudah selesai membuat website dengan HTML dan CSS, langkah selanjutnya adalah menyimpannya di GitHub — supaya bisa diakses dari mana saja, dibagikan ke orang lain, dan bahkan ditampilkan secara live lewat GitHub Pages.

🔧 Yang Dibutuhkan

Akun GitHub gratis di github.com. Tidak perlu install apapun — semua bisa dilakukan langsung dari browser.

Langkah-Langkah Upload ke GitHub

1

Buat Akun GitHub

Buka github.com dan daftar akun gratis. Gunakan email aktif dan buat username yang mudah diingat — ini akan jadi bagian dari URL proyekmu nanti.

2

Buat Repository Baru

Klik tombol hijau New atau tombol + di pojok kanan atas. Isi nama repository dengan huruf kecil semua, tanpa spasi (contoh: ghazy-blog atau website-pribadi). Set visibility ke Public, lalu klik Create repository.

3

Upload File via Drag & Drop

Di halaman repository yang baru dibuat, klik uploading an existing file. Drag dan drop semua file HTML, CSS, dan gambarmu ke area upload. Pastikan semua file ada dalam satu folder yang sama sebelum diupload.

4

Commit (Simpan) File

Setelah semua file ter-upload, scroll ke bawah. Kamu akan melihat kotak Commit changes. Isi pesan singkat (contoh: Upload website pertama), lalu klik tombol hijau Commit changes.

5

Salin Link Repository

Setelah selesai, kamu akan diarahkan ke halaman repository. URL di browser itulah link GitHub kamu — formatnya: github.com/username/nama-repo. Salin URL ini untuk dikumpulkan ke guru.

6

Aktifkan GitHub Pages (Opsional)

Mau website-mu bisa diakses online? Masuk ke Settings → Pages → pilih branch main → klik Save. Dalam beberapa menit, website-mu akan live di username.github.io/nama-repo.

Struktur File yang Benar

Sebelum upload, pastikan struktur folder projectmu seperti ini — semua file HTML harus ada di level yang sama, bukan di dalam subfolder:

Struktur Folder 
nama-repo/
├── index.html       ← halaman utama (wajib ada)
├── artikel.html
├── tentang.html
├── tutorial-html.html
├── tutorial-css.html
└── style.css        ← kalau CSS terpisah
✅ Tips Penting

File utama HARUS bernama index.html (huruf kecil semua). GitHub Pages dan sebagian besar server web otomatis mencari file dengan nama ini sebagai halaman utama.
