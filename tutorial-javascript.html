Mengenal Variabel di JavaScript: var, let, dan const

Ghazy7 April 20265 menit bacaYogyakarta

Kalau HTML adalah struktur dan CSS adalah tampilan, maka JavaScript adalah otaknya sebuah website. Dan hal pertama yang perlu kamu pelajari di JavaScript adalah variabel — tempat menyimpan data. Di JavaScript ada tiga cara mendeklarasikan variabel: var, let, dan const.

💡 Rekomendasi

Untuk kode JavaScript modern, selalu gunakan const secara default. Gunakan let hanya kalau nilai variabel perlu diubah. Hindari var.

Apa itu Variabel?

Variabel adalah kotak penyimpanan untuk data. Kamu bisa menyimpan angka, teks, daftar, atau apapun di dalamnya — lalu memanggilnya kembali kapanpun dibutuhkan.

JavaScript 
// Menyimpan nama
const nama = "Ghazy";

// Menyimpan angka
const umur = 17;

// Menampilkan ke konsol browser
console.log(nama); // Output: Ghazy
console.log(umur);  // Output: 17

const — Nilai Tidak Bisa Diubah

const dipakai untuk nilai yang tidak akan pernah berubah setelah dideklarasikan. Ini adalah pilihan terbaik untuk sebagian besar variabel karena membuat kode lebih aman dan mudah dibaca.

JavaScript 
const kota = "Yogyakarta";
const pi = 3.14;

// ❌ Ini akan error! Tidak bisa diubah.
kota = "Jakarta"; // TypeError!

let — Nilai Bisa Diubah

let dipakai kalau nilai variabel memang perlu diubah nanti — misalnya skor game, hitungan, atau status.

JavaScript 
let skor = 0;
console.log(skor); // 0

skor = skor + 10;
console.log(skor); // 10

skor = skor + 5;
console.log(skor); // 15

var — Cara Lama, Hindari

var adalah cara lama mendeklarasikan variabel di JavaScript (sebelum 2015). Masalahnya, var punya perilaku yang membingungkan bernama "hoisting" dan tidak mengenal batas blok kode. Ini sering menyebabkan bug yang susah dilacak.

⚠️ Peringatan

Jangan gunakan var di kode baru. Kamu mungkin menemukannya di tutorial lama, tapi JavaScript modern sudah punya let dan const yang lebih aman.

Perbandingan Ketiga Variabel

Kata KunciBisa Diubah?Kapan Pakai
const❌ TidakDefault — untuk nilai tetap
let✅ YaNilai yang perlu diubah (skor, counter)
var✅ YaJangan dipakai di kode baru

Contoh Praktis

JavaScript — Contoh Nyata 
// Data yang tidak berubah → const
const namaBlog = "Ghazy Dev";
const tahunDibuat = 2026;

// Data yang berubah → let
let jumlahPengunjung = 0;
jumlahPengunjung++;   // jadi 1
jumlahPengunjung++;   // jadi 2

console.log(`Blog: ${namaBlog}, Pengunjung: ${jumlahPengunjung}`);
// Output: Blog: Ghazy Dev, Pengunjung: 2
