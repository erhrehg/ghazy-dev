Pengenalan Python: Bahasa Pemrograman yang Wajib Dipelajari

Ghazy25 Maret 20269 menit bacaYogyakarta

Python adalah salah satu bahasa pemrograman paling populer di dunia saat ini. Alasannya sederhana: sintaksnya mudah dibaca seperti bahasa manusia, dan bisa dipakai untuk hampir segalanya — dari membuat website, menganalisis data, hingga membangun kecerdasan buatan (AI).

Kenapa Harus Belajar Python?

🌐

Web Development

Dipakai oleh Instagram, Pinterest, dan Spotify untuk backend mereka.

🤖

Kecerdasan Buatan

Bahasa utama untuk machine learning dan AI dengan library seperti TensorFlow.

📊

Analisis Data

Dipakai data scientist untuk mengolah dan memvisualisasikan data besar.

⚙️

Otomasi

Mengotomasi tugas berulang seperti rename file, kirim email, atau scraping web.

Instalasi Python

Sebelum bisa menulis kode Python, kamu perlu menginstall Python di komputermu. Caranya mudah: buka python.org, download versi terbaru, dan install. Pastikan centang opsi "Add Python to PATH" saat instalasi.

💡 Alternatif Online

Tidak mau install? Pakai replit.com atau google.com/colab — bisa menulis dan menjalankan Python langsung di browser, gratis.

Program Python Pertamamu

Tradisi dalam belajar bahasa pemrograman adalah menulis program "Halo Dunia" pertama. Di Python, ini hanya satu baris:

Python 
print("Halo Dunia!")
# Output: Halo Dunia!

print("Namaku Ghazy, dari Yogyakarta")

Variabel di Python

Di Python, kamu tidak perlu menulis const atau let seperti di JavaScript. Cukup tulis nama variabel, tanda sama dengan, dan nilainya.

Python 
# Menyimpan data
nama = "Ghazy"
kota = "Yogyakarta"
umur = 17

# Menampilkan data
print(nama)   # Ghazy
print(kota)   # Yogyakarta
print(umur)   # 17

# Gabungkan teks dengan f-string
print(f"Halo, namaku {nama} dari {kota}")
# Output: Halo, namaku Ghazy dari Yogyakarta

Tipe Data Dasar

Python 
# String (teks)
nama = "Ghazy"

# Integer (bilangan bulat)
nilai = 95

# Float (bilangan desimal)
ipk = 3.85

# Boolean (benar/salah)
lulus = True

# List (daftar)
pelajaran = ["HTML", "CSS", "Python"]
print(pelajaran[0])  # HTML (index mulai dari 0)

Kondisi dan Perulangan

If / Else

Python 
nilai = 85

if nilai >= 90:
    print("Nilai A")
elif nilai >= 75:
    print("Nilai B")
else:
    print("Perlu belajar lebih giat")

# Output: Nilai B

Perulangan For

Python 
bahasa = ["HTML", "CSS", "Python", "JavaScript"]

for item in bahasa:
    print(f"Belajar: {item}")

# Output:
# Belajar: HTML
# Belajar: CSS
# Belajar: Python
# Belajar: JavaScript
✅ Langkah Selanjutnya

Setelah menguasai dasar-dasar ini, coba buat program kalkulator sederhana atau program yang bisa menerima input dari pengguna menggunakan fungsi input().
