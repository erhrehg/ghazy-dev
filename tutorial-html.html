Belajar HTML dari Nol: Panduan Lengkap untuk Pemula

Ghazy 14 April 2026 8 menit baca Yogyakarta

HTML (HyperText Markup Language) adalah bahasa yang digunakan untuk membuat struktur halaman web. Kalau website adalah sebuah rumah, maka HTML adalah fondasi dan dinding-dindingnya. Semua yang kamu lihat di internet — teks, gambar, tombol, form — semuanya dibangun dengan HTML.

💡 Perlu diketahui

HTML bukan bahasa pemrograman. HTML adalah bahasa markup — artinya kita hanya "menandai" bagian-bagian konten, bukan menulis logika atau perhitungan.

Apa itu Tag HTML?

HTML bekerja dengan menggunakan tag. Tag adalah tanda yang memberitahu browser bagaimana menampilkan sebuah konten. Hampir semua tag HTML punya tag pembuka dan tag penutup.

HTML 
<!-- Contoh tag HTML -->
<p>Ini adalah sebuah paragraf.</p>
<h1>Ini adalah judul besar.</h1>
<strong>Teks ini akan dicetak tebal.</strong>

Struktur Dasar Dokumen HTML

Setiap halaman HTML harus punya struktur dasar yang benar. Tanpa struktur ini, browser masih bisa menampilkan kontenmu — tapi tidak akan optimal. Berikut adalah kerangka wajib setiap file HTML:

HTML 
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
  <title>Judul Halaman</title>
</head>
<body>
  <!-- Konten website ada di sini -->
  <h1>Halo Dunia!</h1>
  <p>Ini website pertamaku.</p>
</body>
</html>

Penjelasan setiap bagian:

TagFungsi
<!DOCTYPE html>Memberitahu browser ini adalah file HTML5
<html>Akar / pembungkus seluruh halaman
<head>Berisi informasi halaman (tidak terlihat di browser)
<meta charset>Mengatur encoding karakter (penting untuk huruf Indonesia)
<title>Judul tab di browser
<body>Semua konten yang terlihat oleh pengguna

Tag-Tag HTML yang Paling Sering Dipakai

Ada ratusan tag HTML, tapi kamu tidak perlu hafal semuanya. Cukup kuasai tag-tag ini dulu, dan kamu sudah bisa membangun sebagian besar halaman web:

HTML — Tag Teks 
<!-- Heading: h1 paling besar, h6 paling kecil -->
<h1>Judul Utama</h1>
<h2>Sub Judul</h2>
<h3>Sub Sub Judul</h3>

<!-- Paragraf -->
<p>Ini adalah paragraf biasa.</p>

<!-- Cetak tebal dan miring -->
<strong>Teks tebal</strong>
<em>Teks miring</em>
HTML — Gambar & Link 
<!-- Gambar -->
<img src="foto.jpg" alt="Deskripsi gambar" />

<!-- Link / tautan -->
<a href="https://google.com">Kunjungi Google</a>

<!-- Link ke halaman lain di website yang sama -->
<a href="artikel.html">Lihat Artikel</a>
HTML — List 
<!-- Daftar tidak berurutan (bullet) -->
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<!-- Daftar berurutan (angka) -->
<ol>
  <li>Belajar HTML</li>
  <li>Belajar CSS</li>
  <li>Belajar JavaScript</li>
</ol>

Membuat Halaman Web Pertamamu

Sekarang saatnya praktik. Buka Notepad (Windows) atau TextEdit (Mac), lalu ketik kode berikut ini. Simpan dengan nama index.html, lalu buka filenya di browser.

HTML — Halaman Pertamamu 
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8" />
  <title>Website Pertamaku</title>
</head>
<body>
  <h1>Halo! Namaku Ghazy</h1>
  <p>Ini adalah website pertamaku yang dibuat dengan HTML.</p>

  <h2>Yang Aku Pelajari Sekarang</h2>
  <ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
  </ul>

  <p>Aku dari <strong>Yogyakarta</strong> dan suka coding!</p>
</body>
</html>
✅ Tips

Selalu simpan file HTML dengan ekstensi .html — bukan .txt. Kalau kamu pakai Notepad, pilih "All Files" saat menyimpan, lalu ketik nama file dengan ekstensi .html secara manual.

Langkah Selanjutnya

Selamat! Kamu sudah mengenal dasar-dasar HTML. Sekarang HTML-mu masih polos dan tidak ada warnanya — itu normal. Langkah selanjutnya adalah belajar CSS untuk memberi tampilan dan warna pada HTML yang sudah kamu buat.
