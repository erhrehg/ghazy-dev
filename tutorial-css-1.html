Flexbox vs Grid: Kapan Harus Pakai Yang Mana?

Ghazy10 April 20266 menit bacaYogyakarta

Dua fitur layout CSS yang paling powerful adalah Flexbox dan CSS Grid. Keduanya bisa dipakai untuk mengatur posisi elemen di halaman web — tapi keduanya dirancang untuk tujuan yang berbeda. Banyak pemula bingung harus pakai yang mana, dan akhirnya asal pilih atau malah mencampur keduanya secara sembarangan.

💡 Aturan Praktis

Flexbox untuk satu dimensi (baris ATAU kolom). Grid untuk dua dimensi (baris DAN kolom sekaligus).

Apa itu Flexbox?

Flexbox (Flexible Box) adalah sistem layout CSS yang dirancang untuk mengatur elemen dalam satu arah — baik horizontal (baris) maupun vertikal (kolom), tapi tidak keduanya sekaligus. Flexbox sangat cocok untuk komponen kecil seperti navigasi, tombol, kartu, dan header.

CSS — Flexbox Dasar 
/* Aktifkan Flexbox */
.container {
  display: flex;
  gap: 1rem;
  align-items: center;    /* rata tengah vertikal */
  justify-content: space-between; /* rata kanan-kiri */
}

/* Contoh: navbar */
nav {
  display: flex;
  align-items: center;
  justify-content: space-between;
  padding: 1rem 2rem;
}

Apa itu CSS Grid?

CSS Grid adalah sistem layout dua dimensi — artinya kamu bisa mengatur elemen secara baris DAN kolom sekaligus. Grid cocok untuk layout halaman secara keseluruhan, seperti sidebar + konten, atau grid kartu artikel.

CSS — Grid Dasar 
/* Layout 3 kolom */
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 1.5rem;
}

/* Layout sidebar + konten */
.page-layout {
  display: grid;
  grid-template-columns: 1fr 300px;
  gap: 2rem;
}

Perbandingan Langsung

Flexbox — Gunakan Ketika:

  • Mengatur item dalam satu baris (navbar, tombol)
  • Meratakan elemen secara vertikal di tengah
  • Ukuran item tidak perlu presisi
  • Konten menentukan ukuran elemen
  • Membuat komponen kecil (kartu, badge, form row)

CSS Grid — Gunakan Ketika:

  • Membuat layout halaman utama
  • Butuh kolom dan baris secara bersamaan
  • Ukuran kolom harus konsisten dan presisi
  • Layout menentukan ukuran item
  • Membuat grid foto, kartu artikel, dashboard

Contoh Nyata: Kapan Pakai Masing-Masing

Pakai Flexbox untuk Navbar

HTML + CSS 
<!-- HTML -->
<nav class="navbar">
  <div class="logo">ghazy.dev</div>
  <ul class="nav-links">
    <li><a href="#">Beranda</a></li>
    <li><a href="#">Tutorial</a></li>
  </ul>
</nav>

/* CSS */
.navbar {
  display: flex;              /* ← Flexbox */
  justify-content: space-between;
  align-items: center;
  padding: 1rem 2rem;
}

Pakai Grid untuk Layout Halaman

HTML + CSS 
<!-- HTML -->
<div class="page">
  <main>Konten utama</main>
  <aside>Sidebar</aside>
</div>

/* CSS */
.page {
  display: grid;             /* ← Grid */
  grid-template-columns: 1fr 300px;
  gap: 2rem;
}
✅ Kesimpulan

Flexbox dan Grid bukan saingan — keduanya saling melengkapi. Gunakan Grid untuk struktur halaman besar, dan Flexbox untuk komponen di dalamnya. Keduanya bisa dipakai bersama dalam satu halaman.
